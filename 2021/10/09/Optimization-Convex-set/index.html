    基本集合定义

    • 仿射集

      若通过集合C⊆RnC\subseteq R^nC⊆Rn 中的任意两个同点的直线仍在集合CCC中，则称集合CCC是仿射的

      ∀x1,x2∈C,θ∈R s.t.θx1+(1−θx2)∈C\forall x_1,x_2\in C,\theta \in R\ \\s.t. \quad\theta x_1+(1-\theta x_2)\in C ∀x1​,x2​∈C,θ∈R s.t.θx1​+(1−θx2​)∈C

      可对应定义多个点的情况

      比如线性方程组Ax=bAx=bAx=b的解集XXX就是仿射集，

      反之，任何的仿射集都可以表示成为某个线性方程组的解集

      仿射包

      集合C⊆RnC\subseteq R^nC⊆Rn中的点的所有仿射组合组合成的集合成为CCC的仿射包

      记作aff Caff \ Caff C

      aff C={θ1x1+...+θkxk∣x1,..xk∈C, θ1+...+θk=1}aff\ C=\{\theta_1 x_1 +...+\theta_k x_k | x_1,..x_k\in C ,\ \theta_1 +...+\theta_k =1 \} aff C={θ1​x1​+...+θk​xk​∣x1​,..xk​∈C, θ1​+...+θk​=1}

    • 凸集

      连接集合CCC中的任意两点的线段都在CCC内，则称CCC为凸集

      ∀x1,x2∈C,θ∈[0,1] s.t.θx1+(1−θx2)∈C\forall x_1,x_2\in C,\theta \in [0,1]\ \\s.t.\quad \theta x_1+(1-\theta x_2)\in C ∀x1​,x2​∈C,θ∈[0,1] s.t.θx1​+(1−θx2​)∈C

      显然：仿射集都是凸集

    • 锥

      凸锥的定义如下

      ∀x1,...,xk∈C,θi≥0 s.t.θ1x1+...+θkxk∈C\forall x_1,...,x_k\in C,\theta_i \ge0\ \\s.t.\quad \theta_1 x_1+...+\theta_k x_k\in C ∀x1​,...,xk​∈C,θi​≥0 s.t.θ1​x1​+...+θk​xk​∈C

    • 超平面与半空间

      超平面的表示如下所示，也可看做法线方向为aaa的超平面，常数决定与与原点的偏移

      {x∣aTx=b}\{x|a^Tx =b\} {x∣aTx=b}

      而一个超平面则将RnR^nRn划分为两个闭的半空间，表示如下

      {x∣aTx≤b}\{x|a^Tx \le b\} {x∣aTx≤b}

      易证半空间是凸的，（按照定义即可），但却不仿射

    • 多面体

      多面体可被有限个线性等式与不等式定义

      P={ajT≤bj,j=1...m,cjTx=dj.1...p}P = \{a^T_j\le b_j,j=1...m,c_j^T x = d_j .1...p \} P={ajT​≤bj​,j=1...m,cjT​x=dj​.1...p}

      即多面体是由有限个半空间与超平面的交际。显然多面体是凸集

    • 特殊矩阵集合与半正定锥

      对称矩阵集合

      Sn={X∈Rn×n∣XT=X}S^n = \{X\in R^{n\times n}| X^T = X \} Sn={X∈Rn×n∣XT=X}

      半正定矩阵的集合

      S+n={X∈Sn∣X≥0}S^n_+ = \{X\in S^n| X\ge 0 \} S+n​={X∈Sn∣X≥0}

      正定矩阵的集合

      S++n={X∈Sn∣X>0}S^n_{++} = \{X\in S^n| X> 0 \} S++n​={X∈Sn∣X>0}

      显然有Sn,S+nS^n,S^n_+Sn,S+n​是凸锥，而S++nS^n_{++}S++n​不是凸锥：不包含特殊超平面

    保凸运算

    首先凸集的数乘、加以及任意的交所得到的集合都是凸集

    • 交集

      易证交集运算是保凸的

    • 仿射变换（缩放、平移、投影）

      设fff:Rn→Rm:R^n\to R^m:Rn→Rm 是仿射变换， 即f(x)=Ax+b,A∈Rm×n,b∈Rmf(x) = Ax+b,A\in R^{m\times n},b\in R^mf(x)=Ax+b,A∈Rm×n,b∈Rm

      则有：

      • 凸集在fff下的像是凸集：S⊆RnS\subseteq R^nS⊆Rn 是凸集 ⇒f(S)={f(x)∣x∈S}\Rightarrow f(S) =\{f(x)|x\in S \}⇒f(S)={f(x)∣x∈S}) 是凸集
      • 凸集在fff的原像是凸集:C⊆RnC\subseteq R^nC⊆Rn 是凸集 ⇒f−1(C)={f(x)∣x∈C}\Rightarrow f^{-1}(C) =\{f(x)|x\in C \}⇒f−1(C)={f(x)∣x∈C}) 是凸集

    • 线性分式与透视函数的保凸性

      透视变换保凸性定理：集合{(x,t)∣x∈Rn,t>0}\{(x,t)|x\in R^n,t>0 \}{(x,t)∣x∈Rn,t>0}是凸的，则透视变换得到的集合{xt∣x∈Rn,t>0}\{\frac xt|x\in R^n,t>0 \}{tx​∣x∈Rn,t>0}也是凸集

      分式线性变换的保凸性定理：集合{x∣x∈Rn}\{x|x\in R^n\}{x∣x∈Rn}是凸集，则下面的集合也是凸集

    分离超平面定理

    下面阐述一些凸集的应用：使用超平面或者仿射函数将两个不相交的凸集分离开，其结果便是超平面分离定理。基本解释如下：

    有C,DC,DC,D两个不相交的凸集，则必然存在a≠0,bs.t.∀x∈Ca\not = 0,b \quad s.t. \forall x\in Ca​=0,bs.t.∀x∈C有aTx≤b,∀x∈D,aTx≥ba^Tx\le b,\forall x\in D,a^Tx\ge baTx≤b,∀x∈D,aTx≥b或者说仿射函数aTx−ba^Tx-baTx−b在CCC中非正，在DDD中非负。则可称该超平面为C.DC.DC.D的分离超平面

    其逆定理：存在右侧条件，则两凸集不相交同样适用较广
